# Flutter (.Dart)

## Installation & Running Program :
- ```
  dart pub get
  ```
- ```
  flutter run --no-sound-null-safety
  ```
  
## Export the Program :
### Web Apps (Deploying to Netlify):
- ```
  flutter build web --no-sound-null-safety
  ```
